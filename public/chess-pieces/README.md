# Chess Piece SVG Assets

Place your chess piece SVG files in this directory.

## Required Files

You need 12 SVG files (6 piece types × 2 colors):

**White Pieces:**
- `white-pawn.svg`
- `white-knight.svg`
- `white-bishop.svg`
- `white-rook.svg`
- `white-queen.svg`
- `white-king.svg`

**Black Pieces:**
- `black-pawn.svg`
- `black-knight.svg`
- `black-bishop.svg`
- `black-rook.svg`
- `black-queen.svg`
- `black-king.svg`

## Where to Get Chess Piece SVGs

You can find free chess piece SVG files from:

1. **Wikimedia Commons**: Search for "chess pieces SVG"
   - https://commons.wikimedia.org/wiki/Category:SVG_chess_pieces

2. **Create Your Own**: Use design tools like Figma or Inkscape

3. **Use Unicode Chess Symbols**: As a quick placeholder, you can use text-based chess symbols:
   - ♔ ♕ ♖ ♗ ♘ ♙ (white)
   - ♚ ♛ ♜ ♝ ♞ ♟ (black)

## File Format

- Format: SVG (Scalable Vector Graphics)
- Recommended size: 64x64 or larger (they will be scaled to 56px by default)
- Transparent background preferred
- Files should be named exactly as shown above (lowercase, with hyphens)

## Placeholder

Until you add your SVG files, you might see broken images or missing pieces on the board.
The chess board will still function, but pieces won't be visible.
